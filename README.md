# Fundus Image Classification
This project focuses on classifying retinal (fundus) images into infected and normal categories using deep learning techniques.

##  Model
ResNet18 (Pretrained)

##  Results
- Accuracy: 82.9%
- Confusion Matrix implemented

##  Features
- Image preprocessing
- Deep learning classification
- GPU support (CUDA)

##  Future Work
- Lesion segmentation using UNet

##  Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy
